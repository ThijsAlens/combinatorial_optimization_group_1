# Installation guide
Follow this guide to setup the correct enviorment
## Linux / mac
You should be able to excecute the `setup.sh` file, if not, do excecute the following commands.
1.  **Create a virtual environment:**
    ```python3 -m venv venv```
2.  **Activate the virtual environment:**
    ```source venv/bin/activate```
3.  **Install the necessary packages:**
    ```pip install -r requirements.txt```

## Windows

Please follow these steps manually in your terminal (PowerShell or CMD).

1.  **Create a virtual environment:**
    ```python3 -m venv venv```

2.  **Activate the virtual environment:**
    ```.\venv\Scripts\activate```

3.  **Install the necessary packages:**
    ```pip install -r requirements.txt```
